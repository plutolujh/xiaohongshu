[1.9.0] - 2026-04-08

笔记分享功能

功能: 在笔记详情页添加分享按钮，点击后生成分享图片供用户保存后分享到微信朋友圈

位置: NoteDetail.jsx, NoteDetail.css

实现:

  • 安装 html2canvas 依赖
  • 在笔记详情页添加分享按钮
  • 使用 Canvas API 生成 9:16 竖版分享图片 (375x667)
  • 实现封面图 cover 模式（等比缩放，避免变形）
  • 添加渐变遮罩、文字阴影提升可读性
  • 显示标题、简介、作者、标签、点赞数等信息
  • 底部添加品牌水印「美食笔记」

新增依赖

html2canvas: ^1.4.1

修改文件

  • src/pages/NoteDetail.jsx
  • src/pages/NoteDetail.css
  • package.json
