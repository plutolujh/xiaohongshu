# 项目自动测试指南

## 项目概述

项目名称：小红书美食分享

主要功能：
- 用户注册与登录
- 笔记发布与管理
- 评论功能
- 标签管理
- 用户关注
- 反馈系统
- 图片上传

## 测试环境

- 前端：React 应用，运行在 http://localhost:3000
- 后端：Express 服务器，运行在 http://localhost:3004
- 数据库：Supabase PostgreSQL
- 存储：Supabase Storage

## 测试脚本

### 1. 用户认证测试

#### 1.1 登录测试

**测试步骤**：
1. 访问登录页面：http://localhost:3000/login
2. 输入用户名 "lujh" 和密码 "123456"
3. 点击登录按钮
4. 验证是否成功跳转到首页

**预期结果**：
- 登录成功后跳转到首页
- 页面显示用户头像和昵称

#### 1.2 注册测试

**测试步骤**：
1. 访问注册页面：http://localhost:3000/register
2. 填写注册信息
3. 点击注册按钮
4. 验证是否成功跳转到登录页面

**预期结果**：
- 注册成功后跳转到登录页面
- 显示注册成功提示

### 2. 笔记管理测试

#### 2.1 发布笔记

**测试步骤**：
1. 登录系统
2. 点击顶部导航栏的「发布」按钮
3. 填写笔记标题、内容
4. 上传图片
5. 选择标签
6. 点击发布按钮
7. 验证是否成功跳转到笔记详情页

**预期结果**：
- 笔记发布成功
- 跳转到笔记详情页
- 显示发布的笔记内容

#### 2.2 编辑笔记（测试用户自己发的笔记）

**测试步骤**：
1. 登录系统（使用测试账号 demo/123456）
2. 发布一条新笔记（标题："测试笔记"，内容："这是一条测试笔记"）
3. 进入个人主页
4. 找到刚才发布的笔记
5. 点击编辑按钮
6. 修改笔记内容（例如：将内容改为"这是修改后的测试笔记"）
7. 点击保存按钮
8. 验证修改是否生效

**预期结果**：
- 笔记修改成功
- 跳转到笔记详情页
- 显示修改后的内容
- 个人主页中显示的笔记内容也已更新

#### 2.4 我的笔记显示

**测试步骤**：
1. 登录系统（使用测试账号 demo/123456）
2. 发布1-2条新笔记
3. 进入个人主页
4. 查看"我的笔记"区域

**预期结果**：
- 个人主页显示当前用户发布的所有笔记
- 笔记按时间倒序排列（最新的在前面）
- 每条笔记显示标题、图片、发布时间等信息
- 每条笔记有编辑和删除按钮
- 只显示当前用户自己的笔记，不显示其他用户的笔记

#### 2.5 删除笔记

**测试步骤**：
1. 登录系统
2. 进入个人主页
3. 找到要删除的笔记
4. 点击删除按钮
5. 确认删除
6. 验证笔记是否从列表中消失

**预期结果**：
- 笔记删除成功
- 笔记从列表中消失
- 显示删除成功提示
